Тестовое задание на должность web-программиста в компании BHM 

Тестовое задание необходимо выполнить на фреймворке Yii версии 2.

для установки фреймворка и компонентов используйте composer
результат работы разместите в Git репозитории 

Имея структуру базы данных создайте модели и контроллеры для приложения "интернет магазин" со следующими возможностями

1) просмотр списка категорий с количеством товаров в категории
2) просмотр списка товаров категории  (название, цена, картинка, 300 символов описания )
3) отображение товара с комментариями к нему и 2 рандомными товарами из этой же категории
4) отображения самого комментируемого товара со списком комментариев (и именами пользователей, которые оставили комментарий)
5) просмотр списка брэндов с кол-вом комментариев (оставленных к товарам этих брендов)


Внешний вид тестового приложения не играет роли, можете использовать стандартные стили и виджеты Yii. 
для моделей используйте ActiveRecord. 

В комментариях к выполненному заданию укажите количество часов, затраченные на разработку, укажите самую трудную, по-вашему мнению часть задания.
Старайтесь использовать на полную возможности фреймворка. 

Укажите список  относительных урл по которому можно будет увидеть выполнения заданий
например: 

1)  /categories
2)  /cat-1
3)  /prod-4
...
